## مساله

به یک عدد k رقمی می گوییم آرمسترانگ اگر مجموع توان k ام ارقام آن با خود عدد برابر باشد.
یک عدد n در ورودی به شما داده می شود تعیین کنید n یک عدد آرمسترانگ هست یا خیر (YES یا NO)

محدودیت ها :
`n <= 10^5`

## ورودی نمونه 1

```sh
153
```

## خروجی نمونه 1

```sh
YES
```

توضیحات :
n = 153 : 1^3 + 5^3 + 3^3 = 153

## ورودی نمونه 2

```sh
45
```

## خروجی نمونه 2

```sh
NO
```

توضیحات :
n = 45 : 4^2 + 5^2 = 41
